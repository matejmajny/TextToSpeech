# What can it do?
* Say sentence in english (its a bit robotic)
* Can work offline
* Speed options

# Libraries used
* [pyttsx3](https://github.com/nateshmbhat/pyttsx3) - text to speech library
* [tkinter](https://docs.python.org/3/library/tkinter.html) - GUI library
